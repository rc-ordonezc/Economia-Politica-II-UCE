## 🛠️ Metodología de Documentación

Para garantizar la fidelidad y el rigor académico del contenido, cada módulo se procesa mediante el siguiente flujo:
1. **Captura:** Transcripción de audio de las explicaciones del profesor y registro fotográfico de las pizarras.
2. **Procesamiento:** Limpieza de muletillas, corrección de ortografía técnica y estructuración analítica en formato Markdown con asistencia de modelos de lenguaje (Qwen).
3. **Integración:** Fusión cronológica del texto con esquemas conceptuales y diagramas lógicos extraídos de los mapas visuales.
