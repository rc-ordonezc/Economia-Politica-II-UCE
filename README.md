# Economía Política II - UCE 🏛️📊

¡Bienvenido/a! Este repositorio contiene las transcripciones estructuradas y apuntes académicos correspondientes a las clases de **Economía Política II** en la Universidad Central del Ecuador.

El objetivo de este espacio es sistematizar el contenido teórico y analítico de la materia, transformando las transcripciones de clase y las notas manuscritas en un recurso de estudio digital e interactivo de alta calidad.

## 🛠️ Metodología de Documentación

Cada clase se construye en 6 pasos: (1) **recepción de fuentes** (transcripción + apuntes manuscritos), (2) **limpieza y fusión** de las mismas eliminando muletillas pero conservando conceptos, autores y citas, (3) **estructuración lógica** con párrafos de 4–6 líneas, (4) **notación y formato** en Markdown con fórmulas LaTeX, (5) **enlace cruzado** entre clases, y (6) **revisión final** de coherencia y consistencia notacional.
