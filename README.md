## 🛠️ Metodología de Documentación

Cada clase se construye siguiendo un pipeline reproducible de 6 pasos:

1. **📥 Recepción de fuentes** — transcripción cruda de la clase + apuntes manuscritos.
2. **🧹 Limpieza y fusión** — se eliminan muletillas, se conservan conceptos, autores, citas y ejemplos.
3. **🧠 Estructuración lógica** — cada tema: *introducción → desarrollo teórico → ejemplo/apunte*, con párrafos de 4–6 líneas.
4. **📐 Notación y formato** — símbolos en minúscula (con excepciones `Mt` e `ints`), conceptos clave en negrita, fórmulas en LaTeX.
5. **🔗 Enlace cruzado** — cada clase incluye una sección de *continuidad pedagógica* que apunta a las otras.
6. **🧪 Revisión final** — verificación de consistencia notacional, validación de LaTeX y coherencia con las fuentes.
